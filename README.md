# Pujan Shah — Marketing & Advertising Portfolio

A premium personal portfolio website for Pujan Shah, a Marketing & Advertising professional with 1 year of experience in social media, brand marketing, and campaign execution.

## Tech Stack

- **Framework**: TanStack Start (React 19 + Vite 7)
- **Router**: TanStack Router v1 (file-based routing)
- **Styling**: Tailwind CSS 4 + custom CSS (red & black theme)
- **Content**: Content Collections (markdown)

## Running Locally

```bash
npm install
npm run dev        # starts dev server at http://localhost:3000
npm run build      # production build
```

## Project Structure

```
src/routes/index.tsx   — Full single-page portfolio (Hero, About, Experience, Skills, Case Studies, Gallery, Contact)
public/                — Uploaded campaign images and profile photo
src/styles.css         — Red & black theme, animations, reveal classes
```
